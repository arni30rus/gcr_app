# Changelog


## [1.0.5] - 14-06-2026
Добавлено  

   - Кнопка "Установить пароль", для защиты важных кнопок (сброса БД, выхода из приложения, импорта левой БД).
   - "Интервал сихнронизации" переделан в выпадающий список с возможностью самому вводить время.
   - Добавлена инструкция по развертыванию своего Supabase сервера на Ubuntu.
   - В таблицу клиентов добавлено поле telegram_id, для работы с телеграм ботом.
   - Поле Telegram_id добавлено на форму редактирования клиента.

## [1.0.4] - 9-06-2026
Добавлено

   - Кнопка "Продлить" на экране сканера, появляется при сканировании истекшего абонемента.
   - Таблица renewals (Продление абонементов) для точного подщета продления.
   - Дашборды Продлено сегодня\за месяц. Изменена логика подсчета в дашбордах.
   - Исправлен цвет отображения VIP абонемента при сканировании.
   - Исправлено отображение ФИО в контейнере, ограничено 2 строками.
   - Исправлено отображение порядкового номера клиента, динамически уменьшается шрифт.
    
## [1.0.3] - 7-06-2026
Добавлено

   - Кнопка полного сброса базы данных в настройках (начать с нуля).
   - Подтверждение перед выходом/сменой базы данных.
   - Тип абонемента "Разовое посещение" (флаг is_one_time_visit).
   - Таблица visits (История посещений) для точного подсчета входов.
   - Предупреждение о повторном посещении в течение дня (Anti-pass-back).
   - Инструкция по созданию собственной базы Supabase (обновлена с учетом новых таблиц).

## [1.0.2] - 05-06-2026
Добавлено

   - Экран статистики (Дашборд): активные, новые, истекающие, разовые посещения.
   - Динамические карточки статистики по каждому типу абонемента.
   - Русская локализация календаря.
   - Формат дат ДД-ММ-ГГГГ на всех экранах.
   - Экспорт списка клиентов в CSV.
   - Звуковое оповещение при сканировании (разные звуки для успеха/ошибки).
   - Кнопка включения/выключения звука в сканере.
   - Защита от множественного сканирования одной карты (кулдаун 3 сек).

## [1.0.1] - 01-06-2026
Добавлено

   - Локальный режим работы (полноценная работа без интернета и Supabase).
   - Экспорт/Импорт локальной базы данных (.db файлы).
   - Мягкое удаление (Архив). Удаленные клиенты скрываются из списка, но остаются в БД.
   - Очистка локального архива.
   - Динамические типы абонементов (пользователь сам создает любые типы).
   - Мультиарендность (Залы). Изоляция данных разных залов в одной БД Supabase.
   - Авторизация через Supabase Auth.

## [1.0.0] - 27-05-2026
Добавлено

   - Базовая архитектура (SQLite локально + Supabase облако).
   - Сканер штрихкодов: Камера (Android) + HID-сканер (Windows) + Ручной ввод.
   - Создание, редактирование и поиск клиентов.
   - Проверка активности абонемента (Время, дни недели, Безлимит, VIP).
   - Автоматическая синхронизация с облаком (по таймеру и по кнопке).
   - Сборка под Windows (.exe через Inno Setup) и Android (.apk).
   - Кроссплатформенность (Windows 10/11, Android 10+).
